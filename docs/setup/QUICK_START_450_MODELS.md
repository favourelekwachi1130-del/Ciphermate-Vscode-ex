# Quick Start: 450+ AI Models in 2 Minutes

## Step 1: Get OpenRouter API Key (30 seconds)

1. Go to [openrouter.ai](https://openrouter.ai)
2. Sign up (free account works)
3. Copy your API key from dashboard

---

## Step 2: Configure CipherMate (30 seconds)

1. Open VS Code Settings (Cmd/Ctrl + ,)
2. Search "CipherMate"
3. Set:
   - `AI Provider`  †  `openrouter`
   - `AI > OpenRouter > API Key`  †  Paste your key
   - `AI > OpenRouter > Model`  †  Choose one:
     - `openai/gpt-5` (GPT-5)
     - `anthropic/claude-sonnet-4-20250514` (Claude Sonnet 4.5)
     - `google/gemini-2.0-flash-exp` (Gemini 2.5 Pro)
     - Or any of 450+ models from [openrouter.ai/models](https://openrouter.ai/models)

---

## Step 3: Test (10 seconds)

1. Open CipherMate Chat
2. Type: "Test connection"
3. Should work!    

---

## Done!    ‰

You now have access to:
-     Claude Sonnet 4.5
-     Gemini 2.5 Pro  
-     GPT-5
-     450+ more models

**Switch models anytime in settings - no code changes needed!**

## Want Different Providers?

### OpenAI Direct
- Get key from [platform.openai.com](https://platform.openai.com)
- Set `AI Provider`  †  `openai`
- Enter key in `AI > OpenAI > API Key`

### Anthropic (Claude) Direct
- Get key from [console.anthropic.com](https://console.anthropic.com)
- Set `AI Provider`  †  `anthropic`
- Enter key in `AI > Anthropic > API Key`

### Google Gemini Direct
- Get key from [aistudio.google.com](https://aistudio.google.com)
- Set `AI Provider`  †  `gemini`
- Enter key in `AI > Gemini > API Key`
